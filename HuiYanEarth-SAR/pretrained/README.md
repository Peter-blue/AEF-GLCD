# HuiYan pretrained weights

Pretrained weights are not distributed in this repository. Place the two generator checkpoints here:

```text
HuiYanEarth-SAR/pretrained/huiyan_sar_v1.pth
HuiYanEarth-SAR/pretrained/huiyan_opt_v1.pth
```

Alternatively, pass custom paths with `--pretrained_opt2sar_path` and `--pretrained_sar2opt_path`.
